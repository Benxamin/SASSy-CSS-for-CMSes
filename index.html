How SASS Works

  • UNIX commands
  • Requires a compiler
  • Imports .sass, .scss, .css, .less
  • Converts to a tree structure
  • Outputs to .css

Compiling SASS

From root to specific file paths:

$ sass --watch path/to/sass/styles.sass:path/to/css/styles.css

From assets to css subfolder:

$ sass --watch sass/:css/

In the css folder:

$ sass --watch *

Variables

$var: value
  // Colors
  $green:     #819c31
  $navy:      #012a37
  $orange:    #ed8000
  // Links
  $link: $green
  $focus: #77bdd1
  $hover: #70c1da 
  $visited: #408295
  // Fonts
  $heading: 'CopseRegular',Archer,'Gill Sans',Georgia,sans-serif
  $copy: 'PTSansRegular','Trebuchet MS',Calibri,'Gill Sans',sans-serif
  $bold: 'PTSansBold','Trebuchet MS',Calibri,'Gill Sans',sans-serif
  $italic: 'PTSansItalic','Trebuchet MS',Calibri,'Gill Sans',sans-serif

SASS Mixins

SASS Mixin Syntax

@mixin clearfix()
  overflow: auto
  overflow: -moz-scrollbars-none
  // This makes ie6 get layout
  display: inline-block
  // and this puts it back to block
  &
    display: block
.clearfix
  @include clearfix()

SASS Mixin Syntax

=clearfix
  overflow: auto
  overflow: -moz-scrollbars-none
  // This makes ie6 get layout

  &
    display: block
.clearfix
  +clearfix

SASS Mixin Syntax

$vendors: moz, wekbit, o, ms
=rotate($deg: 90)
    @each $v in $vendors
        #{-#{$v}-transform}: rotate(#{$deg}deg)
.test
    +rotate(22)
.test {
  -moz-transform: rotate(22deg);
  -wekbit-transform: rotate22deg;
  -o-transform: rotate(22deg);
  -ms-transform: rotate(22deg); }

String Interpolation

@mixin vertical-gradient($startcolor, $stopcolor)
  // default bgcolor - averages colors
  background-color: mix($startcolor, $stopcolor)
  // Firefox 3.6+
  background-image: -moz-linear-gradient(top, $startcolor, $stopcolor)
  // Safari 4+, Chrome 1+
  background-image: -webkit-gradient(linear, left top, left bottom, from($startcolor), to($stopcolor))
  // Safari 5.1+, Chrome 10+
  background-image: -webkit-linear-gradient($startcolor, $stopcolor)
  // Opera 11.10+
  background-image: -o-linear-gradient($startcolor, $stopcolor)
  // IE 9
  background-image: -ms-linear-gradient(left bottom, $startcolor, $stopcolor)
  // IE 6+
  filter: progid:DXImageTransform.Microsoft.gradient(
          GradientType=0,StartColorStr=#{$startcolor}, 
          EndColorStr=#{$stopcolor})
